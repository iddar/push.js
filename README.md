push.js
=======

Simple juego multijugador en el que el ganador es quien pulse el boton mas rápida mente.

Herramientas utilizadas:
- Node.js   
- HTML5
- CSS3
- Javascript
- Git/GitHub (Control de versiones)
- Cloud9 (IDE)
- Heroku (Distribución)
